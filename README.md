# 🎬 Netflix SQL Analysis

Este proyecto simula una plataforma tipo Netflix con el objetivo de analizar el comportamiento de usuarios y contenido utilizando SQL.

## 🧠 Objetivo

Explorar y analizar datos para obtener insights como:
- Contenido más visto
- Usuarios más activos
- Géneros más populares

## 🛠️ Tecnologías utilizadas

- PostgreSQL
- SQL

## 🗂️ Estructura del proyecto

- create_tables.sql → creación de tablas
- insert_data.sql → carga de datos
- queries.sql → consultas de análisis

## 📊 Ejemplos de análisis

### 🔥 Contenido más visto
Permite identificar qué títulos tienen mayor cantidad de visualizaciones.

### 👤 Usuario más activo
Muestra qué usuarios consumen más contenido.

### 🎬 Géneros más populares
Analiza tendencias de consumo según género.

## 🚀 Conclusión

Este proyecto demuestra habilidades en:
- Modelado de datos
- Uso de JOINs
- Funciones de agregación (COUNT)
- Análisis de información
